# SmartCourse Feedback Analyzer

A **PHP Web Application** that analyzes student feedback using **Machine Learning (PHP-ML)** and recommends relevant resources based on extracted keywords. This project integrates a library from **Packagist.org** using **Composer**.

##  Features

- Analyze sentiment (Positive, Negative, Neutral) of feedback
- Extract keywords from feedback
- Recommend learning resources
- Simple PHP web interface
- Uses **PHP-ML library** for Machine Learning
